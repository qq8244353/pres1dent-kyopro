# 写像12相
n 個の区別する玉をk 個の区別しない箱に入れる場合の数（全ての箱に1つ以上）は第2種スターリング数として表されます.
第2種スターリング数は以下の二通りで求めることができます.
## 定義から
{{< katex display >}} S(n, k) = \frac{1}{k!}\sum_{i=1}^{k}(-1)^{k-i}{}_kC_ii^n {{< /katex >}}
## 漸化式から
{{< katex display >}} S(n, k) = S(n - 1, k - 1) + k * S(n - 1, k) {{< /katex >}}
