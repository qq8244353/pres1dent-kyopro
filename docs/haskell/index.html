                                <$> と <*> #

                                <$> は fmap の略. <*> は <$> の引数が複数ある時に繋げるために使う.

                                ghci> (*2) <$> [1..3]
[2,4,6]
ghci> (*) <$> [1 .. 3] <*> [11 .. 13]
[11,12,13,22,24,26,33,36,39]

                                replicateM n #

                                replicateM n でn回アクションをおこなって、結果を配列に入れて返す. (Mはモナドのこと) 例:

                                input1 <- replicateM n getInts
input2 <- replicateM n do
    [u, v] <- getInts
    // dosomething
    return (something)
